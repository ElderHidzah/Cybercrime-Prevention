Sentiment Analysis for Cybercrime Prevention

The system connects to the twitter and retrieve tweets.
Allows user to enter any topic
Retrieves tweets related to the topic in real time
Classifies tweets into categories(Positive, Opinion, Factual, Negative)
Supported on Windows OS

Extract the Project folder
Install the requirements in requirements.txt
Open virtual environment
Run the file: run.py
Enter the address 127.0.0.1:5000 on your browser
